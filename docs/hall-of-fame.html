Founding Security Advisor

The highest honor. Reserved for researchers who discover critical vulnerabilities during Origin Fortress's early days (pre-v1.0). Founding Advisors get permanent recognition, a profile link on this page, and acknowledgment in every major release. This title can never be earned again after v1.0.

🏆

Hall of Fame

For any verified security vulnerability report — scanner bypasses, policy engine escapes, audit log tampering, or any other valid security finding. Your name (or handle) and contribution are listed permanently.

🙏

Honorable Mention

For reports that don't qualify as security vulnerabilities but still improve Origin Fortress's security posture — edge cases, hardening suggestions, documentation improvements, or defense-in-depth recommendations.

How It Works

1

Find a Bug

Test Origin Fortress's scanners, policy engine, or any component. Try the Hack Challenge for guided scenarios.

2

Report It

Email security@origin-fortress.com with details, reproduction steps, and impact assessment. See SECURITY.md.

3

Get Verified

We'll acknowledge within 48 hours and assess within 7 days. Valid findings get a CVE if applicable.

4

Get Recognized

Your name goes on the Hall of Fame permanently. Critical findings pre-v1.0 earn the Founding Security Advisor title.
